Una variable es una forma en la cual se puede almacenar informacion para luego poder acceder a esa misma informacion de una manera mas facil y rapida. Para crear una variable se debe utilizar la palabra var , o en su defecto const o let si queremos crear otro tipo de variable , seguido del nombre que le queremos asignar y un signo de igual para decirle que valor queremos que tome.
Un string es un bloque de texto entre " " y al estar entere comillas el programa entiende que no es ninguna propiedad 
Una funcion es una forma de cambiar variables , calcularlas o de sistematizar algun comportamiento con estas mismas. Para comenzar hay que escribir la pabalabra function seguido de parentesis en los cuales iran los argumentos que funcionan para luego reemplazar en la funcion y que tomen esos valores . La unica forma de que accedamos a algo dentro de la funcion es devolverlo es decir usando return ya que el console log no tiene acceso fuera de la funcion . Tambien lo que hace el return es devolver el codigo y una vez que retorna todo lo que este abajo de este mismo no va a ser analizado .
La palabra clave if es para proponer un supuesto es decir que si pasa algo se retorne true pero si eso no sucede se pasa por arriba ese codigo y se ejecutaria la otra condicion que planteamos. 
Los valores booleanos son dos posible valores que determinan si algo es "true" o "false" . Como por ejemplo 1 + 3 = 4 es "true" pero 1 + 3 = 5 es "false".
